Lets start off by creating are delay function. This function will tell our microcontroller to wait x milliseconds before carrying out the next statement to do this drag and drop the "Function No Return" Block from the "Functions" category and name the function "delay".

Create a parameter by clicking the plus symbol found on the top left corner of the "Function No Return" Block. Give the parameter the type of "int" and the name "ms" which is short for milliseconds.
